# Job Cover Letter Generator

This is a Python program that allows you to generate professional cover letters from your resume and job descriptions using OpenAI's GPT API. It can also fetch relevant job postings using the Jooble API.

## Features

- Extracts structured information (skills, education, experience) from a PDF resume using OpenAI
- Searches for relevant job listings using Jooble API
- Generates personalized cover letters using GPT-3.5
- GUI file picker for ease of use

## Prerequisites

- Python 3.8+
- OpenAI API Key
- Jooble API Key
